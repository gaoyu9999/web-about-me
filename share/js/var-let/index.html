Var let

# 示例

在for循环中：在使用var时，整个程序运行到结束用的都是同一个循环变量的值
//使用var声明，得到3个3
var a = [];
for (var i = 0; i < 3; i++) {
  a[i] = function () {
    console.log(i);
  };
}
a[0](); //3
a[1](); //3
a[2](); //3
在使用let时，每次循环时，let定义变量的结果都做了独立模块的解析。let比var在一个块级作用域中的渗透力要强很多，意思就是作用域在向下传递时，整个会穿透这个块的其他子块
//使用let声明，得到0,1,2
var a = [];
for (let i = 0; i < 3; i++) {
  a[i] = function () {
    console.log(i);
  };
}
a[0](); //0
a[1](); //1
a[2](); //2
