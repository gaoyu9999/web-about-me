# 如何使用

nginx-1.23.2\html\airui4 指定目录下放dist（前端打包产物）文件

  1. start nginx ==> 启动
  2. nginx.exe -s stop ==> 停止
  3. nginx.exe -s reopen ==> 重启
  4. nginxexe -s reload ==> 重新加载配置文件

此时启动一定会跨域。
